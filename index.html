Fully automated

The project automates stock market anlysis, investment decisions and buy/sell orders. It builds a model which identifies investment opportunities. When all criteria are met, the script places a real transaction order in an online stock broker.
 

ETL and ML

The system is consisted of multiple processes handling ETL, data analysis, monitoring, evaluation. It pulls current quotes, cleans the data and puts in a database. Generates a number of descriptors based on technical analysis. Rebuilds the descriptor tables to assure their completeness and correctness.
